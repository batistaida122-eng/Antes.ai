Projeto "Estudante Respeitado" (ABNT-AI)
​O Problema
​Quantos estudantes já não foram desencorajados por professores que, em vez de orientar, preferiram o escárnio? A falta de acesso a um suporte digno na formatação de trabalhos acadêmicos não deveria ser uma barreira para ninguém.
​Este projeto nasce da frustração de um aluno que ouviu "pergunte para alguém da sua família" quando buscou ajuda. Este é o nosso basta.
​A Solução
​O ABNT-AI é uma API de código aberto que automatiza a formatação de trabalhos acadêmicos seguindo as normas da ABNT e corrige a gramática em tempo real. O objetivo é simples: permitir que qualquer aluno, independentemente da sua base familiar ou financeira, consiga entregar um trabalho com formatação profissional, sem depender da "boa vontade" de ninguém.
​Por que contribuir?
​Este não é apenas um projeto técnico. É um manifesto contra o elitismo acadêmico. Se você acredita que o conhecimento deve ser acessível e que professores devem ser mentores, e não barreiras, ajude-nos a aprimorar nossas regras de processamento de texto.
​Stack Tecnológica
​Backend: Python (FastAPI)
​Processamento de Texto: LanguageTool
​Regras de Negócio: Algoritmos personalizados para NBR 10520 e NBR 6023.
​Desenvolvido para que nenhum aluno precise ouvir que não tem "família para ensinar" a fazer um trabalho acadêmico.
